# أليثيا - أول ترمينال عربي للبرمجة على أندرويد

أليثيا هي واجهة ترمينال عربية 100%، تدعم أوامر عربية مثل:
- `ع` = ls
- `ن` = cd
- `ط` = pwd
- `عرض` = cat

### الميزات:
- واجهة RTL عربية فصحى.
- تشغيل Node.js وPython محليًا.
- بدون إنترنت.
- مفتوح المصدر.

### التشغيل:
```bash
npm install
npm run dev
